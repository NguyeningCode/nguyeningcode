    2017 - 2018

    Lead Salesforce Developer


    As part of the BizApps team, I built deployment tools for the team using ANT scripts, Bash scripts, and SFDX to improve efficiency and quality of life for developers. I also was responsible for creating the Github repo for our SFDC code which allowed us to do version control better.
